# Summary

This is a filament holder for the newer Creator Pro with the slot-type filament holder.  It was designed to hold filament reels with a 32mm or 50mm center hole (Hatchbox, etc.).  It mounts the same way as the FlashForge holder, and has clips to hold the reel in place.

Added a new longer version that fits the newer Hatchbox spools with the 55x70mm holes.

# Instructions

Designed to be printed with the spool rail horizontal.  Supports for the mounting tabs are required.
